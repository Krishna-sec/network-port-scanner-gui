# 🔍 Network Port Scanner GUI

A multithreaded network port scanner with a graphical user interface (GUI) built using Python and Tkinter.

---

## 🚀 Features
- ⚡ Fast multithreaded port scanning
- 🖥️ User-friendly GUI
- 📡 Scan IP address or hostname
- 🔎 Detect common services (HTTP, SSH, FTP, etc.)
- 📊 Real-time progress tracking
- 💾 Save scan results to file

---

## 🛠️ Technologies Used
- Python
- Socket Programming
- Multithreading
- Tkinter (GUI)
